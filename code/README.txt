===============================
支撑材料说明
===============================

1. 该支撑材料由三个文件夹构成，MATLAB 文件夹包含所有的 Matlab 文件，Python 文件夹包含所有的 Python 文件，以及 Results 文件夹包含所有的结果文件。

2. 第二题与第三题所要求的未知样品的数据文件保存在 Results 文件夹，分别命名为 "result_2.xls" 和 "result_3.xls"。

3. Python 运行须知：
   Python 版本为 3.6.2，GCC 版本为 7.1.1
   依赖库为 numpy, PIL, xlrd, 及内建库 math。在 Windows 10 WSL 环境下运行成功。
   由于使用了相对路径，需要在 Python 目录内运行。

4. Matlab 版本为 Mac 下的 R2016a 版本，在 macOS 10.13 下运行成功。
